🔍 Image Similarity Metrics

📚 Metric Definitions:

SSIM (Structural Similarity Index): Measures how similar the structure, luminance, and contrast are between images. Values range from 0 to 1, where 1 means identical images.

MSE (Mean Squared Error): Calculates the average squared differences between pixel values. Lower values indicate greater similarity, with 0 meaning identical images.

PSNR (Peak Signal-to-Noise Ratio): Measures signal quality in decibels. Higher values indicate better quality/similarity.

Pixel Difference: Shows the percentage of pixels that differ between the two images, providing a straightforward measure of visual change.
